## 文件说明

本目录包含以下原始素材文件：

### 工作流相关文件
- `proposal-chat.txt` - 提案阶段对话交流提示词
- `proposal-clarify.txt` - 提案阶段需求澄清提示词
- `proposal-generate.txt` - 提案阶段文档生成提示词
- `delta-spec-chat.txt` - 增量规格阶段对话交流提示词
- `delta-spec-clarify.txt` - 增量规格阶段规格澄清提示词
- `delta-spec-generate.txt` - 增量规格阶段文档生成提示词
- `delta-design-chat.txt` - 增量设计阶段对话交流提示词
- `delta-design-clarify.txt` - 增量设计阶段方案探讨提示词
- `delta-design-generate.txt` - 增量设计阶段文档生成提示词
- `tasks-chat.txt` - 任务拆分阶段对话交流提示词
- `tasks-clarify.txt` - 任务拆分阶段任务分析提示词
- `tasks-generate.txt` - 任务拆分阶段文档生成提示词
- `validation-chat.txt` - 一致性验证阶段对话交流提示词
- `validation-clarify.txt` - 一致性验证阶段一致性检查提示词
- `validation-generate.txt` - 一致性验证阶段文档生成提示词

### 方法论文件
- `sdd-methodology-guide.md` - SDD 方法论框架，包含各阶段的认知目标、思考框架和质量标准

## 使用说明

本目录中的文件是原始素材，用于：
1. 作为设计要素提取的参考来源
2. 保留原始的提示词格式和内容
3. 便于追溯和验证优化后的内容

这些文件不会被直接使用，而是将其中的设计要素提取并整合到 `references/` 目录下的相应文件中。
