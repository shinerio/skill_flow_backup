Create slides with an engineering blueprint aesthetic meets modern minimal design.
Color System: Pure white background #FFFFFF with cool light gray grid texture #F2F3F5. Primary text in deep charcoal #1A1A1A. Structural lines and decorative elements in muted blue-gray #B8C0CC. A single burnt-orange accent #E8732A used exclusively for key concept highlights, directional arrows, and emphasis markers. Occasional moss green #4A6741 for positive/success indicators only. No gradients, no shadows, flat color throughout.
Layout: Generous whitespace with content occupying 45–55% of each slide. Left-aligned two-column layout for comparison slides, single-column for concept slides. Strict 3-tier information hierarchy: large bold headline → supporting subhead → body text. Bottom footer area carries version number and module name in monospace.
Typography: Ultra-heavy Chinese sans-serif (思源黑体 Heavy or Noto Sans SC Black) for Chinese headlines. JetBrains Mono or similar monospace font for English labels, technical terms, and code references — always lowercase or ALL-CAPS, never mixed case. Lightweight PingFang SC or Noto Sans SC Regular for body text. Title-to-body size ratio approximately 3:1. Strong weight contrast is the primary visual hierarchy tool.
Decorative Elements: Subtle engineering drawing ornaments confined strictly to corners and narrow margins, including small cross-hair markers +, dimension tick arrows ←→, section reference labels SECTION A-A, and a minimal doc-ID block [DOC-ID: XXX] in the top-left corner occupying no more than 12% of slide width. A faint grid or graph-paper texture on the background. These elements function as atmospheric texture only — they must never compete with content.
Diagrams and Icons: All icons in outline style, 2px uniform stroke, no fill or minimal flat fill. Technical diagrams use dashed lines for process flows, solid lines for structural connections. Node shapes: filled black rectangle for master/primary nodes, outlined circles for sub-nodes. Funnel, clock-quadrant, and pipeline illustrations drawn in the same outline aesthetic. Hand-sketched metaphor illustrations (tangled lines vs. ordered parallel lines) used to contrast chaos and order.
Overall Tone: Precise, structured, and intellectually confident. The visual language of an engineer's system manual, not a marketing deck. Every element earns its place. No decorative flourishes beyond the blueprint motif.
